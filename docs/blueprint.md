# **App Name**: Rose Bowl Retreats

## Core Features:

- Premium Landing Page: Visually engaging landing page with scroll animations and bento layouts to showcase property features and amenities.
- Rooms Listing: Display available rooms with detailed descriptions, high-quality images, and pricing. Use toasters to indicate success/failure when adding rooms to a cart.
- Amenities Listing: Showcase all available hotel amenities with descriptions and images.
- Booking Flow: A comprehensive booking system allowing users to select dates, room types, and finalize reservations. Use toasters to indicate booking status.
- Contact Us Form: A contact form for inquiries, feedback, and support requests, with submission confirmation via toaster.
- Customer Testimonials: Display curated testimonials from previous guests to build trust and credibility.
- Interactive Maps: Integrate a map to display the hotel's location and nearby attractions. LLM tool to decide whether a list of nearby attractions would make a useful accompaniment to the map.
- Room Details: Detailed view of each room type with images, amenities, and booking options. Use toasters to indicate success/failure when attempting to book from the details page.
- Gallery Section: A visually appealing gallery to showcase the hotel's ambiance and facilities through high-quality images.

## Style Guidelines:

- Primary color: Soft gold (#CBB38E) to evoke classic elegance.
- Background color: Light beige (#F5F5DC), providing a warm and inviting feel.
- Accent color: Deep burgundy (#800020) to complement the classic elegance.
- Headline font: 'Playfair' serif for a fashionable and elegant style.
- Body font: 'PT Sans' sans-serif to pair with 'Playfair', ensuring comfortable readability.
- Use simple, elegant icons for navigation and key features.
- Employ a bento layout for the landing page to showcase key features in an organized and visually appealing way.
- Implement subtle scroll animations on the landing page for a modern touch.
